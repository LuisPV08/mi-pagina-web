Programación Estructurada

Variables

Las variables son contenedores para almacenar datos.

Condicionales

Permiten tomar decisiones en el código.

Bucles

Repiten un bloque de código mientras se cumpla una condición.

Funciones

Bloques de código reutilizables que realizan una tarea específica.

Ejemplo de Bucle


                    for (let i = 0; i < 5; i++) {
                        console.log(i);
                    }

Programación Orientada a Objetos

Clases y Objetos

Las clases son plantillas para crear objetos que encapsulan datos y comportamiento.

Herencia

Mecanismo que permite a una clase heredar propiedades y métodos de otra clase.

Polimorfismo

Capacidad de un objeto para tomar diferentes formas y comportamientos.

La POO permite organizar el código en objetos y clases. Aquí verás un ejemplo de una clase en JavaScript:


                class Persona {
                    constructor(nombre, edad) {
                        this.nombre = nombre;
                        this.edad = edad;
                    }
            
                    saludar() {
                        console.log(`Hola, mi nombre es ${this.nombre} y tengo ${this.edad} años.`);
                    }
                }
            
                const persona1 = new Persona('Juan', 30);
                persona1.saludar();
